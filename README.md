# Payment Platform

Payment Platform is an engineering laboratory focused on modern backend architecture and cloud-native development.

It is **not** intended to become a real payment gateway.

The goal is to learn and demonstrate:

- Clean Architecture
- Event Driven Architecture
- AWS
- Docker
- ECS
- PostgreSQL
- Redis
- Amazon SQS
- Outbox Pattern
- Idempotency
- Observability
- CI/CD
